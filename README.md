# 🛒 Zepto Sales Analysis Dashboard | Power BI

> **An Interactive Power BI Dashboard for Analyzing Zepto Sales Performance, Customer Ratings, City-wise Sales, and Business Trends.**

---

## 📌 Project Overview

The **Zepto Sales Analysis Dashboard** is an interactive Business Intelligence solution developed using **Microsoft Power BI**. This dashboard provides valuable insights into sales performance, customer ratings, city-wise sales distribution, and yearly sales trends through dynamic visualizations and interactive filters.

It enables business users and decision-makers to monitor key performance indicators (KPIs), identify high-performing cities, and analyze sales patterns for better strategic planning.

---

## 🎯 Project Objectives

- Analyze overall sales performance.
- Track yearly sales trends.
- Identify top-performing cities.
- Monitor customer ratings.
- Provide interactive filtering for better analysis.
- Build a clean and professional Business Intelligence dashboard.
